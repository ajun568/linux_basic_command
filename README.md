# linux基础命令
> 这是一篇关于Linux基础命令的文档 想系统学习Linux命令的出门左拐
* 创建文件夹
```
mkdir folder
```
* 展示列表信息
```
ls 列表文件名
ls -a 列表文件名 <包括隐藏文件>
ll 列表文件详情信息
```
* 查看当前所在位置
```
pwd
```
* 进入文件夹
```
cd folder
cd .. <返回上一文件夹>
```
* 删除操作
```
rm -rf folder <删除文件夹>
rm file_name <删除文件>
```
* 拷贝
```
cp file_name1 file_name2 <拷贝file_name1并命名为file_name2>
```
* 重命名
```
mv file_name1 file_name2
```
* 移动
```
mv file_name folder/file_name <把file_name移动到folder文件夹下>
```
